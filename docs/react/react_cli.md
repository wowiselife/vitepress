## 创建 React 项目

你可以使用 create-react-app 命令来创建一个新的React应用程序。

```
npx create-react-app my-react-app

```

这将创建一个名为 my-react-app 的新目录，并在其中初始化一个新的React应用程序。

## 进入项目目录

```
cd my-react-app

```

## 启动开发服务器

```
npm start

```
