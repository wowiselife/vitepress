## 创建 vue 项目

```
npm init vite@latest

```

- Project name: my-vite-vue-app
- Package name: my-vite-vue-app
- Select a framework: vue
- Select a variant: javascript

## 进入项目目录

```
cd my-vite-vue-app

```

## 安装依赖

```
npm install

```

## 启动开发服务器

```
npm run dev
```
