## 安装 Vue CLI

Vue CLI 是一个官方提供的用于快速搭建 Vue.js 项目的工具。你可以使用 npm 在全局安装 Vue CLI。在终端中执行以下命令：

```
npm install -g @vue/cli

```

## 创建 Vue 项目

```
vue create my-vue-app

```

## 进入项目目录

```
cd my-vue-app

```

## 启动开发服务器

```
npm run serve

```

备注：即使选择的是 vue3 的版本，默认的代码还是 vue2 的选项式写法
